Rotas

A primeira coisa que o UI5 faz é carregar um "index.html", que por sua vez vai carregar a biblioteca do UI5.

Após esse ponto nenhuma request é feita para carregar uma tela, o UI5 usa um link interno de URL com o "#" (hashtag) para ir para outra view, permitindo assim que o usuário cadastre a URL nos favoritos.
Internamente, o UI5 cadastra no histórico essa navegação para emular o comportamento de navegação típico, habilitando navegar no histórico.
